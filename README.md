# 2042_sensor_testing

Для запуска на новом устройстве:
1. Перейти в папку catkin_ws. В командной строке ввести "catkin_ws".

Добавление датчика
1. Добавить датчик в sensor library {Имя датчика, название мира, название тестов и результаты}
2. Добавить файл с датчиком в resources/{sensor_type}/
3. Добавить мир в resources/worlds/ Или использовать уже существующий мир.
4. В test_manager создать функцию с тестом и добавить её в словарь test_functions

Пока работают только camera/model и camera/model1. Проверьте, что прога с ними работает.
