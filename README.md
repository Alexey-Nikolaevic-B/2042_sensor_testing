# 2042_sensor_testing

Для запуска на новом устройстве:
1. Перед запуском перейти в папку catkin_ws. В командной строке ввести "catkin_make".
2. Выйти обратно в корневую папку проекта и запустить ./main

Добавление датчика
1. Добавить датчик в sensor_library.py {Имя датчика, название мира, название тестов и результаты}
2. Добавить файл с датчиком в resources/{sensor_type}/
3. Добавить мир в resources/worlds/ Или использовать уже существующий мир.
4. В test_manager создать функцию с тестом и добавить её в словарь test_functions

Пока есть только camera/model и camera/model1. Проверьте, что прога с ними работает.
